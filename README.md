# AboutMeSite (Flask)

Полноценный веб-сайт на Flask с кейсами, формой обратной связи и защищенной админ-панелью.

## Функционал

- Главная страница с hero-блоком, описанием и 5 кейс-примерами.
- Раздел кейсов с отдельной детальной страницей для каждого кейса.
- Форма обратной связи (имя, email, телефон, тема, сообщение) с валидацией.
- Админ-панель с авторизацией и управлением заявками:
  - просмотр всех обращений;
  - отметка как "прочитано";
  - удаление заявок.
- Логирование ключевых событий (вход, выход, новые заявки, действия в админке).

## Технологии

- Flask
- Flask-Login
- Flask-SQLAlchemy
- Flask-WTF (CSRF + формы)
- Jinja2
- SQLite
- Bootstrap 5

## Настройка .env

Все ключи и константы конфигурации хранятся в `.env`.

Скопируйте пример и при необходимости измените значения:

```bash
cp .env.example .env
```

Для Windows PowerShell:

```powershell
Copy-Item .env.example .env
```

## Быстрый запуск

1. Установите зависимости:

```bash
pip install -r requirements.txt
```

2. Запустите приложение:

```bash
python app.py
```

После запуска сайт будет доступен по адресу:

- [http://127.0.0.1:5000](http://127.0.0.1:5000)

## Доступ в админ-панель

- URL: `/admin/login`
- Логин: `admin`
- Пароль: `admin123`

Рекомендуется сменить `DEFAULT_ADMIN_PASSWORD` и `SECRET_KEY` в `.env` перед production-запуском.

## Продакшен-домен

Для домена `aboutme.dimonrond.ru` настройте прокси-сервер (например, Nginx) и WSGI-сервер (gunicorn/uwsgi), затем укажите корректные DNS-записи.
